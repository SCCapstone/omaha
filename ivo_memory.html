iVotronic Memory Module

2-Megabyte Flash Memory Chip

AMD's AM29LV160DT-90EC

Datasheet: U1_AM29LV160DT.pdf

This memory module works in tandem with the other memory module and four other identical memory chips soldered directly to the motherboard to provide data storage and data backup for the information used and generated by the iVotronic during normal operations. This particular chip provides 16 megabits (2 megabytes) of flash memory, and the pins that seat onto the motherboard are just one-to-one extensions of the chip's pins.
