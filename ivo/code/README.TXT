The files contained in this zip file are the executable and source code
referenced in the Intel Application Note AP-720, entitled:

           "Programming Flash Memory through the Intel386(tm)EX 
                    Embedded Microprocessor JTAG Port"

TAPLOADR.EXE will run on the EV386EX Evaluation Board and will program 
the onboard flash through the custom JTAG TAP interface described in 
the application note.

TAPLOADR.CPP is the source code for the .EXE file which was compiled using 
MS Visual C++ version 1.5. See comments in the source code for questions 
about its usage.

README.TXT (this file) contains a desription of the contents of TAPLOADR.ZIP
which is distributed on Intel's America's BBS.
