iVotronic Video Assembly

The iVotronic's video assembly is the component responsible for generating the machine's graphical user interface. The Epson graphics processor uses data it receives to generate the various components to display (shapes, text, and colors) which are sent to the LCD Transmitter that then encodes the information for display.

In addition to displaying information to the user, the display is also touch enabled, and while the driver assembly is the only interface between the motherboard and the display, we know that the Epson processor cannot process user input directly (as no mention is made on the chip's datasheet).Therefore, either ES&S is having the chip process input information ad-hoc, or the user input is simply being ferried through, unaltered, to the machine's CPU.

One design idiosyncrasy worth mentioning is that the assembly board has space on the underside for up to two memory chips; however, those slots were empty on our board.
